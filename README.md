                                     Lion Run
                              
                          
        Rules:

1.You (Lion) need to collect meat, every meat is 10 points.

2.The hunter will try to catch you, just run from him.

3.The hunter collect meat as well, every meat is -5 points.


        Features:

1.Buttons game.

2.Sensor game.

3.Top ten list.

4.Google map.

5.Sound.

6.Glide.

and more.
